HOW TO SAVE PROGRESS ON GAMES
To save your progress your first going to need to make sure the game says ruffle before fully loaded.
If it says ruffle then right click, if it says WAFLASH as the page title, then it saves automatically.

Once here, click the save manager, then you can download your save file and re-import it from here.
